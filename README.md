# Garbage-Detection-AI
Detects the material of garbage from the image uploaded using YOLOv8 trained AI.

### Installation : 
1. Download all the files (app.py, requirements.txt, and best.pt)
2. Put them in 1 directory
3. Open cmd from said directory
4. Run `pip install -r requirements.txt` 
5. Wait for every requirements to install
6. Run `streamlit run app.py`
